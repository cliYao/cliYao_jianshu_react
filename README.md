## jianshu_react

## 说明
使用 React 仿写了一个简单的简书, 纯前端的项目, 部分数据使用 easy-mock 模拟 (easy-mock 响应有时会超级慢~ ) , 还有一部分数据来自维基百科, 客户端需要可以直接访问 Wikipedia 服务器, 才能预览全部功能　
项目完成时间仓促, 存在一些 bug , 欢迎大家帮忙指正, 多谢~
